# Student Management App

## Project Description
This is an Android application developed using Java and Firebase Realtime Database to store student details.

## Features
- Add Student Details
- Firebase Realtime Database
- Input Validation
- Simple User Interface

## Technologies Used
- Java
- XML
- Android Studio
- Firebase
- GitHub

## How to Run
1. Open the project in Android Studio.
2. Connect Firebase.
3. Run the app on an emulator or Android device.
